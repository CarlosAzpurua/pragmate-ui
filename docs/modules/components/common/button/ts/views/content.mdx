# Buttons

El componente de botón se utiliza para crear elementos interactivos que permiten a los usuarios realizar acciones
específicas, como enviar formularios, confirmar selecciones o desencadenar eventos en la interfaz.

### Import :

<Code>{`import { Button } from 'pragmate-ui/components';`}</Code>

### Bordered

Para implementar las variantes bordeadas es necesario agregar el atributo `bordered` al componente.

<BorderedExample />

### Iconos

Para agregar un icono al boton, puedes hacerlo pasando la propiedad `icon`, esta propiedad puede recibir una cadena de
texto, la cúal debe corresponder con el nombre de los <Link href='/components/icons-list'>iconos existentes en la
libreria</Link> o puede recibir un objeto de tipo `IPragmateIcon` Para agregar iconos al botón, agrega el atributo icon
y el icono que desees utilizar.

<IconsExample />

### Tooltips :

Podemos agregar funciones Tooltips a los **Buttons** agregando una propieadad _title_ y el titulo que queramos.

<TooltipsExample />

#### Important :

Para implementar el funcionamiento del tooltips es importante agregar un archivo sass en el template.
