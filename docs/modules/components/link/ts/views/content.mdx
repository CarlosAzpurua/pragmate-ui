# Link

El componente Link Garantiza el redireccionamientos de url al dar click en el, permitiendo agregar al atributo href la
url deseada. Si desea ejecutar una función antes de la redirección, puede pasar un onClick que se ejecutará antes de la
redirección.

### Import :

<Code>{`import { Link } from 'pragmate-ui/components';`}</Code>

Si desea ejecutar una función antes de la redirección, puede pasar un onClick que se ejecutará antes de la redirección.

### Implementation :

<Code>{`<Link href='/' >This is a link to home</Link>`}</Code>
