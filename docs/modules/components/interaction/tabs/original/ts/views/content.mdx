# Tabs

El componente `Tabs` permite a los usuarios navegar entre diferentes secciones de contenido utilizando pestañas. El
componente Tabs es especialmente útil para aplicaciones web y de escritorio que requieren presentar información en
secciones separadas y accesibles. Muestra una lista de pestañas para navegar entre diferentes secciones de contenido.
Personalización de estilos y configuraciones. Soporte para navegación por pestañas, arrastrando y gestos táctiles. Puede
mostrar cualquier tipo de contenido en una pestaña utilizando slots.

### Import :

<Code>{`import { TabsContent, TabsContainer, Tabs } from "pragmate-ui/tabs";`}</Code>

Si pasa la propiedad `thedata-path` la pestaña hará un `routing.replaceState` en el valor pasado a la `thedata-path`.
