# opinions

El componente `Opinions` es un componente de React que se utiliza para mostrar una sección de testimonios o comentarios
de usuarios. El componente utiliza el paquete pragmate-ui/swiper-component para implementar un carrusel que permite
desplazarse entre los diferentes testimonios.

Para usar el componente, debes importarlo en tu archivo y luego renderizarlo en tu componente principal. El componente
recibe una prop llamada `opinion` , que es un arreglo de objetos que contiene las propiedades necesarias para generar
varias instancias del componente.

Cada objeto del arreglo debe tener las siguientes propiedades :

-   `name` : El nombre del usuario que dejó el testimonio.

-   `src` : La URL de la imagen de perfil del usuario.

-   `profession` (opcional): La profesión o cargo del usuario.

-   `comment` : El comentario o testimonio del usuario.

-   `alt` (opcional): El texto alternativo para la imagen de perfil.
