# Spinner

El componente `Spinner` es un elemento de la interfaz de usuario que se muestra cuando se está cargando o realizando una
acción que requiere tiempo, como la carga de datos o la ejecución de una operación en segundo plano. El componente
spinner puede incluir animaciones, colores y estilos personalizados para adaptarse a la apariencia y el estilo de la
aplicación. Personalización de estilos y configuraciones. Integración fácil con diferentes marcos de JavaScript, como
Angular, Vue.js y React. Compatibilidad con diferentes navegadores y dispositivos.
