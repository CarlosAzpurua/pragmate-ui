# ThemeSwitch

Este componente `Theme Switch` es un elemento de la interfaz de usuario que permite cambiar fácilmente entre temas
claros y oscuros en una aplicación o sistema operativo. Este componente puede ser especialmente útil en aplicaciones web
y de escritorio, así como en sistemas operativos como Windows 10, que admiten temas claros y oscuros. Funcionalidades
Muestra un botón o interruptor para cambiar entre temas claros y oscuros.
