# Color Guide

Este documento describe las variables de color utilizadas en la aplicación. Cada variable tiene un propósito específico
en el diseño general.

## Colores Primarios

-   `--primary`: Color principal utilizado en componentes y elementos destacados.
-   `--primary-container`: Fondo para contenedores que usan el color primario.
-   `--on-primary`: Color de texto e iconos sobre el color primario.
-   `--on-primary-container`: Color de texto e iconos sobre el contenedor primario.
-   `--inverse-primary`: Variante invertida del color primario.

## Colores Secundarios

-   `--secondary`: Color secundario, utilizado en elementos secundarios y complementarios.
-   `--secondary-container`: Fondo para contenedores que usan el color secundario.
-   `--on-secondary`: Color de texto e iconos sobre el color secundario.
-   `--on-secondary-container`: Color de texto e iconos sobre el contenedor secundario.

## Colores Terciarios

-   `--tertiary`: Color terciario, utilizado en elementos adicionales y detalles.
-   `--tertiary-container`: Fondo para contenedores que usan el color terciario.
-   `--on-tertiary`: Color de texto e iconos sobre el color terciario.
-   `--on-tertiary-container`: Color de texto e iconos sobre el contenedor terciario.

## Colores de Superficie

-   `--surface`: Color de fondo general para componentes de superficie.
-   `--surface-dim`: Variante más oscura de la superficie.
-   `--surface-bright`: Variante más clara de la superficie.
-   `--surface-container-lowest` a `--surface-container-highest`: Escala de colores para contenedores de superficie.
-   `--surface-variant`: Variante del color de superficie.
-   `--on-surface`: Color de texto e iconos sobre la superficie.
-   `--on-surface-variant`: Color de texto e iconos sobre la variante de superficie.
-   `--inverse-surface`: Color invertido de superficie.
-   `--inverse-on-surface`: Color de texto e iconos sobre la superficie invertida.

## Otros Colores

-   `--background`: Color de fondo general de la aplicación.
-   `--on-background`: Color de texto e iconos sobre el fondo.
-   `--error`: Color utilizado para errores y advertencias.
-   `--error-container`: Fondo para contenedores que usan el color de error.
-   `--on-error`: Color de texto e iconos sobre el error.
-   `--on-error-container`: Color de texto e iconos sobre el contenedor de error.
-   `--outline`: Color utilizado para bordes y líneas divisorias.
-   `--outline-variant`: Variante del color de contorno.
-   `--shadow`: Color utilizado para sombras.
-   `--surface-tint-color`: Color de tinte para la superficie.
-   `--scrim`: Color utilizado para efectos de oscurecimiento o velo.
