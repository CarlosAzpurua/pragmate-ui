# Guía de Colores

Pragmate UI es un sistema que aprovecha los principios del Diseño Material para generar estilos coherentes y
consistentes en toda la plataforma. Utilizando una combinación de variables predefinidas y directrices de diseño
flexibles, Pragmate UI ofrece un enfoque simplificado para estilizar componentes, superficies y contenedores.

## Estructura de Colores

### Superficies

Las superficies son objetos materiales que contienen contenido. El color aplicado define el fondo del componente.

-   `--surface`: Color para superficies materiales.

### Contenedores

Los contenedores agrupan contenido relacionado, con ligeras variaciones para diferenciar elementos.

-   `--primary-container`: Color para elementos clave que necesitan más énfasis.
-   `--on-primary-container`: Color de texto o ícono en el contenedor primario.
-   `--secondary-container`: Color para partes menos enfatizadas o complementarias.
-   `--on-secondary-container`: Color de texto o ícono en el contenedor secundario.
-   `--tertiary-container`: Color para elementos terciarios.
-   `--on-tertiary-container`: Color de texto o ícono en el contenedor terciario.
-   `--info-container`: Color para contenedores de información.
-   `--warning-container`: Color para contenedores de advertencia.
-   `--error-container`: Color para contenedores de error.
-   `--success-container`: Color para contenedores de éxito.

### Colores de Texto

-   `--text-color`: Color de texto principal.
-   `--altern-text-color`: Color de texto alternativo.
-   `--secondary-text-color`: Color de texto secundario.

### Colores Primarios y Secundarios

-   `--primary`: Color primario para énfasis.
-   `--on-primary`: Color de texto o ícono en el color primario.
-   `--secondary`: Color secundario para menos énfasis.
-   `--on-secondary`: Color de texto o ícono en el color secundario.

### Variables Adicionales

-   `--font-family`: Define la fuente utilizada.
-   `--error`: Color para mensajes de error.
-   `--on-error`: Color de texto o ícono en el error.
-   `--background`: Color de fondo.
-   `--on-background`: Color de texto o ícono en el fondo.
-   `--outline`: Color de contorno.
-   `--on-surface`: Color de texto o ícono en la superficie.
-   `--surface-variant`: Color de variante de superficie.
-   `--on-surface-variant`: Color de texto o ícono en la variante de superficie.
-   `--background-color`: Color de fondo general.
