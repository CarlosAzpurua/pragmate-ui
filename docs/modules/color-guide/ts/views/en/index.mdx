# System Colors Documentation

## Pragmate UI Content Generation

Pragmate UI is a system that leverages the principles of Material Design to generate coherent and consistent styles
across the platform. Using a combination of predefined variables and flexible design guidelines, Pragmate UI provides a
streamlined approach to styling components, surfaces, and containers.

## Colors Structure

### Surfaces

Surfaces are material objects that contain content. The color applied defines the background of the component.

-   `--surface`: Color for material surfaces.

### Containers

Containers group related content together, with slight variations to differentiate elements.

-   `--primary-container`: Color for key elements that need more emphasis.
-   `--on-primary-container`: Text or icon color on primary container.
-   `--secondary-container`: Color for less emphasized or complementary parts.
-   `--on-secondary-container`: Text or icon color on secondary container.
-   `--tertiary-container`: Color for tertiary elements.
-   `--on-tertiary-container`: Text or icon color on tertiary container.
-   `--info-container`: Color for information containers.
-   `--warning-container`: Color for warning containers.
-   `--error-container`: Color for error containers.
-   `--success-container`: Color for success containers.

### Text Colors

-   `--text-color`: Primary text color.
-   `--altern-text-color`: Alternative text color.
-   `--secondary-text-color`: Secondary text color.

### Primary and Secondary Colors

-   `--primary`: Primary color for emphasis.
-   `--on-primary`: Text or icon color on primary color.
-   `--secondary`: Secondary color for less emphasis.
-   `--on-secondary`: Text or icon color on secondary color.

### Additional Variables

-   `--font-family`: Defines the font family used.
-   `--error`: Color for error messages.
-   `--on-error`: Text or icon color on error.
-   `--background`: Background color.
-   `--on-background`: Text or icon color on background.
-   `--outline`: Outline color.
-   `--on-surface`: Text or icon color on the surface.
-   `--surface-variant`: Variant surface color.
-   `--on-surface-variant`: Text or icon color on surface variant.
-   `--background-color`: General background color.
