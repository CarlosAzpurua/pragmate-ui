## Custom Properties

Por definicion **Pragmate UI** utiliza una paleta de colores y estilos para los componentes.
Las siguientes variables se utilizan para definir la paleta de colores y estilos en tus componentes.

- **--text-color** : Este es el color de texto principal que se utiliza en tus componentes.
- **--altern-text-color** : Este es el color de texto secundario que se utiliza en tus componentes.
- **--secondary-text-color** : Este es el color de texto secundario que se utiliza en tus componentes.
- **--font-family** : Esta es la fuente tipográfica que se utiliza en tus componentes.

### Colores primarios

Los siguientes son colores que se utilizan para definir el color primario en tus componentes:

- **--primary** : Este es el color principal que se utiliza en tus componentes.
- **--on-primary** : Este es el color de texto que se utiliza encima del color primario en tus componentes.
- **--primary-container** : Este es el color de fondo que se utiliza para los contenedores principales en tus componentes.
- **--on-primary-container** : Este es el color de texto que se utiliza encima del color de fondo primario en tus contenedores principales.

### Colores secundarios

Los siguientes son colores que se utilizan para definir el color secundario en tus componentes:

- **--secondary** : Este es el color secundario que se utiliza en tus componentes.
- **--on-secondary** : Este es el color de texto que se utiliza encima del color secundario en tus componentes.
- **--secondary-container** : Este es el color de fondo que se utiliza para los contenedores secundarios en tus componentes.
- **--on-secondary-container** : Este es el color de texto que se utiliza encima del color de fondo secundario en tus contenedores secundarios.

### Colores terciarios

Los siguientes son colores que se utilizan para definir el color terciario en tus componentes:

- **--tertiary** : Este es el color terciario que se utiliza en tus componentes.
- **--on-tertiary** : Este es el color de texto que se utiliza encima del color terciario en tus componentes.
- **--tertiary-container** : Este es el color de fondo que se utiliza para los contenedores terciarios en tus componentes.
- **--on-tertiary-container** : Este es el color de texto que se utiliza encima del color de fondo terciario en tus contenedores terciarios.

### Otros colores

Los siguientes son otros colores que se utilizan en tus componentes:

- **--error** : Este es el color que se utiliza para indicar un error en tus componentes.
- **--error-container** : Este es el color de fondo que se utiliza para los contenedores de error en tus componentes.
- **--on-error-container** : Este es el color de texto que se utiliza encima del color de fondo de error en tus contenedores de error.
- **--background** : Este es el color de fondo principal que se utiliza en tus componentes.
- **--on-background** : Este es el color de texto que se utiliza encima del color de fondo principal en tus componentes.
- **--surface** : Este es el color de superficie que se utiliza en tus componentes.
- **--on-surface** : Este es el color de texto que se utiliza encima del color de superficie en tus componentes.
- **--outline** : Este es el color que se utiliza para los contornos y bordes en tus componentes.
- **--surface-variant** : Este es un tono más claro del color de superficie principal que se utiliza en algunos casos especiales.
- **--on-surface-variant** : Este es el color de texto que se utiliza encima del color de superficie variante en tus componentes.
- **--info-container** : Este es el color de fondo que se utiliza para los contenedores de información en tus componentes.
- **--warning-container** : Este es el color de fondo que se utiliza para los contenedores de advertencia en tus componentes.
- **--success-container** : Este es el color de fondo que se utiliza para los contenedores de éxito en tus componentes.
