# simply-ui

simply-ui is a collection of reusable components here you can see what they are and how to import and implement them.

## To install

`npm i @bgroup/ui`

## Components

- Button
- Textarea
- Switch
- Input
- Checkbox
- Radio
- Toast

- Modal
- Alert Modal
- Confirm Modal
- Image
- IconButton
- Icon
- Tabs
- Input Controller
- Swiper
- Alert
- Verification code
- DragAndDrop
- Dropdown
- DropdownSidebar
- ThemeSwitcher
- Code
